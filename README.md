# Лабораторные работы по учебной дисциплине «Численные методы»

Реализация базовых вычислительных алгоритмов, методов аппроксимации, отделения корней и решения систем линейных уравнений с использованием языка программирования **PascalABC.NET** и электронных таблиц **MS Excel**.

---

## 📌 О проекте
Данный репозиторий содержит комплекс лабораторных работ, направленных на изучение и практическое применение численных методов. Каждый проект включает в себя математическое описание метода, разработку блок-схемы алгоритма, написание программного кода и анализ погрешностей в зависимости от используемых типов данных (`single`, `real`, `double`, `extended`).

* **Разработчик:** Сартбаев Ж. Б.  
* **Организация:** КГАПОУ «Камчатский политехнический техникум» (КПТ)  
* **Язык программирования:** PascalABC.NET  
* **Инструменты анализа:** MS Excel  

---

## 📂 Содержание и структура лабораторных работ

### Лабораторная работа №1: Приближенное значение величины. Абсолютная и относительная погрешность
* **Цель:** Изучение теории погрешностей и округления чисел.
* **Практика:** Разработка программы для вычисления абсолютной и относительной погрешности приближений, а также исследование реальной точности встроенных математических функций (на примере $sin(x)$) для различных вещественных типов данных.

### Лабораторная работа №2: Оценка погрешности значения функции
* **Цель:** Расчет надежных значащих цифр при выполнении арифметических операций.
* **Практика:** Построение блок-схем и циклических программ для табулирования функций и оценки накопленной погрешности вычислений.

### Лабораторная работа №3: Определение погрешности элементарных функций
* **Цель:** Нахождение значений выражений методом границ и методом строгого учета абсолютных погрешностей после каждой операции.
* **Практика:** Написание программ на Паскале для автоматического вычисления погрешностей сложных тригонометрических и логарифмических выражений.

### Лабораторная работа №4: Отделение корней уравнений
* **Цель:** Изучение этапа локализации действительных корней на заданном отрезке $[a; b]$.
* **Практика:** Реализация алгоритма сканирования отрезка с заданным шагом $h$ для уравнений вида $lg(x) + 6 = x^2$ и $x \cdot sin(x) - 1 = 0$.

### Лабораторная работа №5: Сравнение методов по скорости сходимости итерационного процесса
* **Цель:** Анализ эффективности и экономичности итерационных процедур.
* **Практика:** Программная реализация и сравнение метода половинного деления (дихотомии), метода простых итераций и метода касательных (Ньютона) с заданной точностью $\epsilon = 10^{-6}$.

### Лабораторная работа №6: Графический метод отделения корней в MS Excel
* **Цель:** Визуализация трансцендентных уравнений для нахождения начальных приближений.
* **Практика:** Построение графиков функций в MS Excel, определение точек пересечения графиков (абсцисс корней) и оценка абсолютной погрешности графического метода.

### Лабораторная работа №7: Решение систем линейных алгебраических уравнений (СЛАУ)
* **Цель:** Реализация прямых методов линейной алгебры.
* **Практика:** Программирование схемы единственного деления (метода Гаусса) с прямым и обратным ходом. Вычисление векторов невязок $\mathcal{E}_i$ для оценки точности решения при использовании типов `single`, `real` и `double`.

## 🚀 Как запустить программы

1. Установите среду [PascalABC.NET](https://pascalabc.net/).
2. Склонируйте репозиторий на локальный компьютер.
3. Откройте интересующий файл с расширением `.pas` в среде разработки.
4. Нажмите **Выполнить** (клавишу `F9`) для запуска программы.
5. Для ознакомления с графическими расчетами откройте соответствующие отчетные файлы в формате `.xlsx` через MS Excel.
